# Maritime Logistics Management System

This project is a comprehensive solution for managing maritime logistics, including ship management, route planning, cargo tracking. Built with React and Vite for the front end, and Node.js with Express for the back end, it aims to streamline the operations of maritime shipping. Key features include:

- **Ship registration and status tracking**
- **Route definition and planning**
- **Cargo management and allocation**
- **Document management for shipping operations**
